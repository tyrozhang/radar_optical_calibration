# 雷达-光电联合标定整改方案

## 背景

当前标定系统存在三个核心隐患：

1. **坐标转换忽略了站心系旋转**：BLH 直接算方位角，未经过 ECEF→ENU 转换，长距离时方位角误差可达 0.5°~1°
2. **雷达标定忽略了距离误差传递**：焦距计算受雷达测距误差 1:1 影响，未对 `dR` 进行标定与补偿
3. **光电标定对近处目标过于敏感**：`<300m` 样本几何敏感且多径干扰严重，按距离倒数加权会放大近场噪声

---

## 整改一：坐标系转换（最高优先级，硬错误）

**目标**：彻底消除因坐标系近似导致的系统性方位角偏差。

**方案**：

1. **统一中间坐标系**：所有几何计算必须以 **ECEF（地心地固）** 为跳板，禁止直接对 BLH 做平面近似
   - 步骤1：雷达站 BLH → ECEF（使用 WGS-84 椭球参数）
   - 步骤2：目标 BLH → ECEF
   - 步骤3：以雷达站为原点，建立 ENU 站心坐标系（旋转矩阵由雷达站纬度/经度决定）
   - 步骤4：在 ENU 中计算真值 Az/El：`atan2(E, N)`，`atan2(U, sqrt(E²+N²))`

2. **代码检查点**：全局搜索所有 `blhToAzEl` 或类似函数，确认其内部实现是否包含 ECEF 转换。如果没有，立即标记为高危

3. **验证方法**：取一个远距离样本（>10km），分别用"旧方法（平面近似）"和"新方法（ECEF→ENU）"计算方位角，差值应 `<0.01°`。如果差值 `>0.1°`，说明旧代码存在系统性偏差

---

## 整改二：雷达误差模型扩展（中优先级，精度瓶颈）

**目标**：将距离误差纳入联合标定，避免其向光电引导链传递。

**方案**：

1. **扩展标定向量**：将标定参数从 `X = [dAz, dEl]` 扩展为 `X = [dAz, dEl, dR, dE]`（增加距离偏差、高程偏差，视传感器类型而定）

2. **分阶段标定策略**：
   - **阶段A（角度标定）**：使用远距离目标（>2km），此时距离误差对角度影响极小，独立解算 `dAz, dEl`
   - **阶段B（距离标定）**：在角度标定完成并补偿后，使用标准反射器或 GPS 真值，解算 `dR = R_radar - R_true`

3. **引导时补偿**：
   - 雷达原始输出：`Az_m, El_m, R_m`
   - 补偿后输出：`Az_c = Az_m - dAz`, `El_c = El_m - dEl`, `R_c = R_m - dR`
   - 光电焦距计算必须使用 `R_c`，而非 `R_m`

4. **异常处理**：如果 `|dR| > 阈值`（如 3%），触发告警，提示雷达可能存在故障或气象折射异常，禁止用于精密引导

---

## 整改三：标定数据筛选与加权（高优先级，鲁棒性）

**目标**：消除近场异常样本对全局标定结果的污染。

**方案**：

1. **硬性距离门限**：
   - 标定数据准入条件：`d_min ≤ R ≤ d_max`
   - 建议 `d_min = 300m`（规避多径和几何敏感区）
   - 建议 `d_max = 0.8 × 雷达最大有效距离`（规避信噪比恶化区）

2. **加权策略替换**：
   - 废除 `w = 1/R` 或 `w = 1/R²`
   - 改用 **等权** 或 **基于重投影误差的自适应加权**：
     ```
     w_i = 1 / σ_i²,  其中 σ_i 为第 i 个样本的重投影残差
     ```
   - 或者简单可靠的 `w = 1`（在样本分布均匀时，等权往往比盲目加权更鲁棒）

3. **异常值剔除（RANSAC / 3σ 准则）**：
   - 先用全部样本做最小二乘拟合
   - 计算每个样本的残差 `r_i = |Az_mea - Az_model|`
   - 剔除 `r_i > 3·median(r)` 的样本，迭代至收敛

4. **几何分布要求**：标定航迹不应只在一个方向。要求样本覆盖方位角 `≥180°`、俯仰角覆盖低/中/高多个波段，避免病态几何

---

## 整改四：系统级验证方案（验收用）

**目标**：整改后必须量化证明问题已解决。

| 验证项 | 方法 | 通过标准 |
|--------|------|----------|
| 坐标转换精度 | 取远距离真值点，对比平面近似与 ENU 结果 | 方位角差异 `<0.01°` |
| 标定内符合精度 | 用标定数据做交叉验证（留一法） | Az/El 残差 RMS `<` 雷达标称精度 |
| 标定外符合精度 | 用独立检查点（未参与标定）验证 | 残差 RMS `< 1.5×` 标称精度 |
| 引导精度 | 雷达补偿后引导光电，统计脱靶量 | 脱靶量 `< 1/2` 视场角（或满足任务指标） |
| 近场剔除有效性 | 对比"全量数据"与"剔除<300m"的标定结果 | 后者外符合精度提升 `>20%` |
